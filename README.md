The Bank Management System is a Python-based application developed to simulate core banking operations using a lightweight JSON-based data storage system. This project focuses on backend logic and file handling concepts, allowing users to manage bank accounts and transactions through a simple, menu-driven interface.

The application is designed to replicate basic real-world banking workflows such as account management, deposits, withdrawals, and transaction tracking without using a traditional database.
